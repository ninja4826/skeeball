Start button pressed
  lower release
  wait
  raise release
  play chime 1

point switch hit
  points equal to hole allocation added
  play chime 2
  ball count + 1
  if ball count is nine, check if points are higher than highscore.
  if points are higher than high score, replace
  call reset if ball count is nine

reset button pressed
  call reset

reset
  ball count = 0
  points = 0
